# Vendor Performance Analysis

This project focuses on **evaluating vendor performance** through **data-driven insights**. It involves **cleaning and analyzing procurement datasets**, identifying **top-performing vendors**, and visualizing key metrics such as **purchase contribution, sales performance, procurement, inventory turnover and impact of bulk purchasing**. The insights generated support **strategic procurement decisions and business intelligence initiatives**.
